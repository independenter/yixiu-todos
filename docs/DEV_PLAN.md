## 风险与备注

- **进程检测跨平台差异大**：Linux 读 `/proc`，Windows 用 WMI/ETW，macOS 用 `libproc`。阶段 2.2 优先做 Linux。
- **冲突算法 O(n²)**：n<500 时无感；超过需改区间树 / 扫描线。
- **Tauri 2 移动端成熟度**：iOS/Android 后端复用，但通知/托盘行为差异大，1.x 再投入。
- **强制规则的法律/伦理**：纯本地工具无问题；若上云需告知用户。
